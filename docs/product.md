# Product

## Premise

Video feeds optimize for continued watching. Viuwu optimizes for deliberate choice. A person
saves a search such as "tiny desk concerts" or "weeknight ceramics" and that search becomes a
channel in their guide.

## Product Principles

- The user chooses every channel.
- The guide explains why each video is present: it matched a named search.
- Channel order is explicit and editable.
- YouTube supplies videos but does not decide the guide.
- Calm completion beats infinite scroll.

## First Milestone

The home screen presents a familiar TV guide made from saved searches. Each row has a channel
identity and a finite set of recent matches. Remote focus makes the current choice unmistakable.

The Channels screen previews management controls for adding, pausing, and reordering searches.
Settings exposes the connected YouTube account and playback preferences without hiding them.

First launch requires connecting YouTube through Google's TV device authorization flow. The TV
shows a QR code that opens Google's account selection and consent page with the device code
prefilled, plus a manual code fallback. After connection, the user creates named search channels.
Viuwu persists those channels, loads current YouTube results, and opens selected videos in YouTube.

## Platform

Viuwu is a YouTube application. Saved searches, explicit channel order, and a finite guide remain
the product boundary.
